Seamless javascript code coverage

Blanket.js is an easy to use, flexible, and reliable library which will give your code coverage results without heavy configuration

Download beta 0.9.0

Try-it now and give us feedbacks

Download

The only file you need is blanket.js, dowload the file and add a reference to your usual test runner.

<scrip src="blanket.js"></scrip>

Configure

Add an attribute data-test for each file you want to cover

<scrip src="my-library.js" data-test></scrip>
<scrip src="my-tests-1.js"></scrip>
<scrip src="my-tests-2.js"></scrip>

Enjoy

Run your test as usual and enjoy the report appended to your test results



Support

We currently support Qunit, Jasmine and Mocha. Give us feedback if you encounter some errors. It is still in beta :)
